🖥️ CLI Commands & Flags
Core
gemini — start interactive REPL
-p "prompt" — one-shot non-interactive
-i — stay interactive after -p
-r — resume last session
gemini update — update CLI to latest
Pipe Input
cat file.txt | gemini -p "summarize"
cat file.txt | gemini -p "summarize"
Get-Content file.txt | gemini -p "summarize"
Key Flags
-m flash — choose model/alias
-o out.txt — write output to file
--approval-mode yolo
-w — git worktree mode experimental
-s name — named session
-d — debug / verbose output
--list-sessions — show saved sessions
--delete-session id
--include-directories — include dirs in context
-e KEY=VAL — env var override
--allowed-mcp-server-names a,b
🤖 Model Selection
Alias → Model ID
AliasResolves to

Approval Modes
Ask before every actiondefault
Auto-approve file editsauto_edit
Plan first, then executeplan
Auto-approve everythingyolo
Shift+Tab cycles approval mode in the REPL
Examples
gemini -m flash -p "explain this"
gemini --approval-mode auto_edit
