# Gemini CLI Cheat Sheet

A single-page visual reference for the [Gemini CLI](https://github.com/google-gemini/gemini-cli) — covers everything you need at a glance.

## What's included

- **Keyboard Shortcuts** — navigation, editing, and action keys
- **App Controls** — theme, sandbox, and UI toggles
- **Interactive Commands** — slash commands, session controls, prompt prefixes
- **CLI Commands & Flags** — `gemini` flags, pipe input, one-shot mode
- **Model Selection** — aliases (`pro`, `flash`, `flash-lite`) and model IDs
- **Memory & Context** — `GEMINI.md` hierarchy, memory files
- **MCP Servers** — adding and configuring Model Context Protocol servers
- **Extensions** — custom extensions and how to load them
- **Skills & Agents** — built-in and custom skills/agents
- **Key Settings** — important config options

## Usage

Just open `index.html` in your browser — no build step, no dependencies.
